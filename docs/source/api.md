# API Reference

This section provides detailed documentation of the MCP Server's API.

## MCP Server

The LLM Agent has access to the MCP Server, which provides data retrieval, summarization, visualization, and analysis tools and resources from the USA-NPN. The server is designed to support phenologists and developers in accessing and managing phenological data with natural language and explore it efficiently with an AI Agent.

### Tools, Resources, and Prompts

The MCP Server provides the following tools and resources for developers and phenologists:

- **Tools**: APIs for data retrieval and summarization, visualization, other analysis.
- **Resources**: Access to queried data and structured summaries of literature.
- **Prompts**: Predefined templates for workflows and interacting with LLM agents in phenology.

Refer to the [User Guide](user_guide.md) for usage examples and tutorials. Stay tuned for updates in the [News](news.md) section.
