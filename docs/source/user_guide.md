# User Guide

This guide provides step-by-step instructions for using the MCP Server.

## Getting Started

1. **Setup Server and Client**: The GitHub repository [README.md](../../README.md) contains instructions for setting up the MCP Server locally in Claude Desktop.
2. **Use the Server!**: The MCP Server can be used by Claude in Claude Desktop. The server is designed to be user-friendly and provides a variety of tools and resources for phenologists and developers using natural language querying and analysis with LLM agents.

## Examples

1. **Basic Queries**: Examples of basic queries for retrieving phenological data.
2. **Mapping**: Examples of generating maps using the retrieved data.

Refer to the [API Reference](api.md) for detailed documentation.
